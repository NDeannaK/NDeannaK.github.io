Education

I am currently pursuing my Diploma in IT where I learned various skills whether it be technical, electronic, coding, communications or such.

Most of the subject consist of coding classes where I learned various programming languages such as C, C#, C++, Python, Java, PHP, HTML, CSS, JavaScript.

There is also a database subject where we learned how to use MySQL which I implemented in my mini project to create a login database.

While studying for my Diploma I of course learned the basics of computers, learning how to setup Windows OS and Linux OS where I used a virtual computer to help simulate the process.

I took 2 classes for Business Communication to help improve my communication and documentation process which will be useful in a work environment.

My approach to learning has always been to experience it hands-on, when there is something that I'm passionate about I will try my best to understand it to the fullest and the best way for me to learn things is simply by just doing it.

I have found myself to understand a topic better when I can engage in the experience whether it be my hobbies or when studying. I can do work on little guidance as long as I can try and experience it, which will allow me to do my best to enhance my skills.

Which is why I have chosen to study IT a subject that have many different options of hands-on experiences. While studying I have found myself to enjoy coding. The fact that I can create a system using only my computer is fascinating but the fact that I can work on my skill at any time allowed me to put my all into the projects that I have set for myself.
